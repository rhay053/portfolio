총 4개

  • 검사 종료

    검사가 끝나면 자동으로 심전도 데이터가 전송되나요?

    처방된 검사 기간이 종료되면 자동으로 데이터가 클라우드 서버로 전송 됩니다.

    만약 자동으로 전송이 되지 않은 경우, 모바일 앱 내 ‘데이터 전송’ 버튼을 눌러 전송합니다. 

  • 검사 종료

    검사 종료 후 앱을 바로 삭제해도 되나요?

    의료진이 데이터를 확인하기 전까지는 앱을 삭제하지 마시고, 의료진에게 검사결과를 안내 받은 후에 앱을 삭제하실 것을 권장합니다. 

  • 검사 종료

    심전도 기록은 어떻게 전송하나요?

    온라인 모드로 테스트 진행 후 설정한 시간까지 검사를 마치면 검사기록은 자동으로 전송됩니다.

    만약 검사를 조기 종료한 경우에는 앱에서 ‘데이터 전송’ 버튼을 눌러 전송을 진행해주세요. 

  • 검사 종료

    심전도 데이터가 어디로 전송되나요?

    검사 종료 후 암호화되어 클라우드 서버로 자동 전송되며 지능화된 분석 소프트웨어가 1차 분석 후 S-Patch 웹으로 전송됩니다.

    의료진은 부여된 ID로 S-Patch 웹에 로그인하여 분석된 심전도 데이터를 확인하고 편집할 수 있습니다.
